# Antigravity Demo Steps

Do not rely on Antigravity to browse Kaggle.

1. Open the repo in Google Antigravity.
2. Show `specs/scholarproof_system_spec.md`.
3. Show `AGENTS.md`.
4. Ask Antigravity to inspect the UI.
5. Ask Antigravity to fix one UI issue.
6. Show the diff before accepting.
7. Run the app locally.
8. Use Antigravity browser testing to verify the UI.
9. Show no YOLO auto-approve.
10. Show the final app workflow:
    - Profile.
    - Find Scholarships.
    - Eligibility Checker.
    - Evidence Panel.
    - Draft Clarification Email.
    - Saved Results.
